Отличия от версии 1.0

- описана комманда \TotalNum{} для задания кол-ва билетов вместо стандартной комманды
  изменения значения счетчика \setcounter. Теперь для того, чтобы задать нужное кол-во
  билетов необходимо в преамбуле написать комманду \TotalNum{20} вместо 
  \setcounter{TotalNum}{20}
